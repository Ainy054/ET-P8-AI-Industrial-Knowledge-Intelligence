# AI for Industrial Knowledge Intelligence

## Problem Statement
Build an AI-powered assistant that helps users retrieve and understand information from industrial documents using Retrieval-Augmented Generation (RAG).

## Features
- Multi-PDF document upload
- AI-powered question answering
- FAISS vector database
- HuggingFace embeddings
- Google Gemini 2.5 Flash
- Streamlit interface

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace
- Google Gemini

## Run Locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run app.py
```
